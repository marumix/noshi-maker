# のし紙メーカー

Windows 向けののし紙作成・印刷ツールです。縦書きテキストをドラッグで自由に配置し、PDF保存や直接印刷ができます。

## 機能

- 表書き・社名・役職・氏名の縦書き配置
- ドラッグ＆ドロップで位置・サイズを調整
- 水引のプレビュー表示（印刷には含まれません）
- システムフォント一覧から選択可能
- のし紙データの保存・読み込み（JSON形式）
- 手差しトレイへの自動給紙設定（pywin32使用時）

## 対応用紙サイズ

| 名称 | サイズ |
|------|--------|
| 美濃版 | 394×267mm |
| 半紙版 | 337×245mm |
| 切手版 | 260×185mm |
| 豆のし | 195×145mm |

## 必要環境

- Windows 10 / 11
- Python 3.8 以上
- Pillow（初回起動時に自動インストール）

オプション（印刷トレイの自動設定に使用）:
```
pip install pywin32
```

## 使い方

```bash
python noshi_maker.py
```

1. 表書き（例：御祝）を選択または入力
2. 氏名・社名・役職を入力
3. プレビューで要素をクリックして選択し、ドラッグで位置を調整
4. 「のし紙を保存」でデータを保存、または「印刷」で直接印刷

## 印刷時の注意

- 用紙の向き：**横向き**
- 給紙：**手差しトレイ（トレイ5）**
- 印刷サイズ：**実際のサイズ（100%）**
- 用紙のセット：裏返し・上辺を手前にしてセット
